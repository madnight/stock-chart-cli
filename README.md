# Usage
```bash
# install
git clone https://github.com/madnight/wallstreet
cd wallstreet
peotry install

# run
poetry run python main.py
```

# Config

The list of stocks can be configured via the config.ini file.
```ini
[Symbols]
stocks = AAPL,MSFT,GOOGL,INTC,AMD,PEP,MU,TSLA,NFLX,DIS,AMZN,SPY,QQQ
```
